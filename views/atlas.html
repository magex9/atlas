Atlas

The atlas is an knowledge base of all the development products that we use. This large collection of information represents all of the applications and code that we maintain.

This product belongs to the tools business line which contains internal tools required by IT. This major release was started in 2014.

Modules

Atlas Service
The atlas service is used to manage the business functionality with the database.
Atlas Reporting
The atlas reports combine rdf data to gather metrics from the system.
Atlas UI
The atlas UI is a swing application for viewing the data.
